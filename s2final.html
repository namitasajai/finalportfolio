Section 2 Final Project: Abstract Clock

Donut Clock
Section 2 Final Project
As part of the Computational Science and Web Design project, we delved into the unique history of these disciplines intertwined with the user's system clock. Following a structured process that encompassed brainstorming, sketching, prototyping, and construction, we aimed to create a browser-based design using p5.js that harnessed the three basic events of the system clock: seconds, minutes, and hours. In my contribution, I crafted a visually engaging "donut clock" concept, deviating from the conventional clock design. Within this interactive clock, the flavor of the donut changes every hour, a bite is taken from the donut every minute, and a sprinkle is added to the donut every second. This inventive approach allowed for a dynamic and whimsical representation of time, showcasing my creativity and proficiency in leveraging the system clock's events to create an engaging user experience.
