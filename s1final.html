Impossible Quiz and Webring
Section 1 Final Project | Section 2 Assignment 1
This project involves the creation of an innovative online game that takes inspiration from "The Impossible Quiz", a nostalgic favorite. The website consists of five unique pages, each featuring an imaginative challenge to navigate to the next page. The page invites users to use hints and easter eggs placed around the page to decipher solutions to various tasks. If the user fails to complete a page within the time limit, they are rerouted to the "404 Page". The website effectively integrates a fun navigational toolset into its design, all while adhering to the 5MB per page limit.

Section 2 Assignment 1 involves the creation of a webring, a collection of websites linked together in a circular structure. The webring is a simple way to navigate between a small, independent group of websites. The webring is a great way to discover new websites and explore the work of other creators. For this project, I linked the first page of my Impossible Quiz to the first page of a classmate's Section 1 Final Project. The link is held within the response to the initial prompt on the first page of the Impossible Quiz. Clicking "No" instead of "Yes" prompts the user to the first page of the webring.
